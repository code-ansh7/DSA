# Day 4 - Largest and Second Largest Element

## Topics Learned

### Largest Element
- Finding the maximum element in an array
- Traversing the array and updating maximum value

### Second Largest Element
- Finding the second largest distinct element
- Two Loop Approach
- Single Loop Approach

## Key Concepts

- Maximum element can be found in one traversal.
- Second largest element can be found using:
  - Two loops
  - Single loop
- Sorting is not required.
- Duplicate largest values must be handled carefully.

## Time Complexity

### Largest Element
O(n)

### Second Largest (Two Loops)
O(n)

### Second Largest (Single Loop)
O(n)

3. Second Largest Element (Single Loop)
