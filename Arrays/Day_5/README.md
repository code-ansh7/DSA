## Topics Learned

### Check Sorted Array
- Checking whether an array is sorted or not
- Comparing adjacent elements
- Identifying order violations

### Strictly Increasing Array
- Difference between sorted and strictly increasing arrays
- Handling duplicate elements

## Key Concepts

- Sorted Array allows equal elements.
- Strictly Increasing Array does not allow equal elements.
- Adjacent elements are compared to check order.
- Condition Checking Pattern is used.
